# Kootaney Photography

A single-page editorial site for Loumicha Lapointe.

## Files
- `index.html` — the site
- `tweaks-app.jsx` — design-time tweak panel (palette + font swaps)
- `tweaks-panel.jsx` — tweak panel UI shell

## Photos
The site references photos at `photos/<filename>.jpg`. Place your images in
a `photos/` folder at the repo root (the filenames already match the ones
in your existing `Images/drive-download.../` folder).

## Publishing on GitHub Pages
1. Push these files to your repo
2. Settings → Pages → Source: deploy from branch `main` / root
3. Visit `https://<username>.github.io/<repo>/`
